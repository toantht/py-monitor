# py-monitor
A Basic GUI monitoring with PySimpleGUI

![screenshot](https://raw.githubusercontent.com/toantht/py-monitor/main/Screenshot.png)


## Install

```shell
$ git clone https://github.com/toantht/py-monitor
$ pip install -r requirements.txt
```

## Run

```shell
$ python3 monitor.py
```
